# Opensource Team 6

## 프로젝트 소개

Opensource Team 6 저장소는 팀원들의 자기소개 페이지를 포함한 웹 프로젝트를 관리하기 위한 공간입니다. 각 팀원은 자신의 HTML 소개 페이지를 작성하고, 팀 전체가 협업하여 하나의 프로젝트를 구성합니다.

---

## 팀원

* 애진 (aejin)
* 진근 (jingeun)
* 창민 (changmin)
* 예림 (yerim)

---

## 파일 구성

* **aejin.html**: 애진 소개 페이지
* **changmin.html**: 창민 소개 페이지
* **jin.html**: 진근 소개 페이지
* **yerim.html**: 예림 소개 페이지
* **index.html**: 팀 메인 페이지

---

## 주요 기능

* 각 팀원 개별 소개 페이지 제공
* 메인 페이지에서 팀원 소개 페이지로 이동 가능
* GitHub 기반 협업 및 버전 관리

---

## Git 사용 규칙 (예시)

1. 각자 브랜치를 생성해 작업한 후 main에 merge
2. main에 push하기 전 반드시 최신 버전 pull
3. 충돌 발생 시 팀원과 협의하여 해결
4. 커밋 메시지는 명확하게 작성

---

## 프로젝트 실행 방법

1. 저장소를 클론합니다.

   ```bash
   git clone <git clone https://github.com/aejin-cloud/Opensource_team-6.git>
   ```
2. HTML 파일을 브라우저에서 열어 확인합니다.

---

## 협업 방법

* GitHub Issues를 활용한 작업 관리
* Pull Request 기반 코드 리뷰
